---
description: Enviar alterações para o GitHub com confirmação
---

1. Verifica o status dos arquivos
```
git status
```

2. Adiciona todos os arquivos modificados
```
git add .
```

3. Solicita a mensagem do commit e realiza o commit
// turbo
4. Envia para o repositório remoto (GitHub)
```
git push
```

Dica: Sempre revise o `git status` antes de confirmar o envio!
